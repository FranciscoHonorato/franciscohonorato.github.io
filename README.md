# Portfólio

Este projeto é um portfólio pessoal desenvolvido com HTML, CSS e JavaScript. Ele apresenta meus principais projetos, habilidades e informações de contato.

## Funcionalidades

- Menu responsivo para navegação.
- Alternância de tema claro/escuro.
- Seção de projetos com descrição e links.
- Informações de contato.

## Como executar

Basta abrir o arquivo `index.html` em seu navegador.
